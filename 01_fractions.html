Egyptian Fractions I¶

Background¶

Preliminary Steps¶

Proposition 1.

Given integers \(1 < a < b \in \mathbb{N}\) where \((a,b) = 1\), then

\(\exists\) some \(y > x > 0 \in \mathbb{N}\) where \((x,y) = 1\) and \(x \le a\), \(y \le b\)

which satisfy the equation \(ay - bx = 1\).

Proof:

Having proved the existence of \(x,y \mathbb{N}\) satisfying ..., then

Given vulgar fraction \(\large\frac{a}{b} \normalsize\in \mathbb{Q}\) where \(1 < a < b \in \mathbb{N}\) and \((a,b) = 1\), then

\(\exists\) some \(y > x > 0 \in \mathbb{N}\) such that \((x,y) = 1\) and \(x \le a\), \(y \le b\)

satisfying the equation :math:` largefrac{a}{b} normalsize = largefrac{x}{y} plus ` \(\large\frac{ a \times y - b \times x }{ b \times y }\).

\(ay - bx = 1 \Longrightarrow y = ( bx + 1 ) / a\) and \(x = ( ay -1 ) / b\) ; or

\((bx + 1) \equiv 0 \bmod{a}\) and \((ay - 1) \equiv 0 \bmod{b}\)

Example

For \(a = 23\) and \(b = 47\), find distinct denominators \(d_n\)

\(\large\frac{23}{47} \normalsize =\) \(\large\frac{1}{d_1} \normalsize +\) \(\large\frac{1}{d_2} \normalsize +\) \(\large\frac{1}{d_3} \normalsize +\) \(\large\frac{1}{d_4} \normalsize +\) \(\large\frac{1}{d_5} \normalsize\)

Step 1:

\(\large\frac{23}{47} \normalsize =\) \(\large\frac{1}{47 \times 45} \normalsize +\) \(\large\frac{22}{45} \normalsize\) and \(\large\frac{22}{45} \normalsize =\) \(\large\frac{1}{3} \normalsize +\) \(\large\frac{7}{45} \normalsize\)

Step 2:

\(\large\frac{7}{45} \normalsize =\) \(\large\frac{1}{45 \times 13} \normalsize +\) \(\large\frac{2}{13} \normalsize\)

Step 3:

\(\large\frac{2}{13} \normalsize =\) \(\large\frac{1}{7} \normalsize +\) \(\large\frac{1}{13 \times 7} \normalsize\)

Result:

\(\large\frac{23}{47} \normalsize =\) \(\large\frac{1}{47 \times 45} \normalsize +\) \(\large\frac{1}{3} \normalsize +\) \(\large\frac{1}{45 \times 13} \normalsize +\) \(\large\frac{1}{7} \normalsize +\) \(\large\frac{1}{13 \times 7} \normalsize\)

Method:¶

  1. Find the largest fraction \(\large\frac{x}{y}\) where \(ay - bx = 1\) and \((x,y) = 1\).

  2. Suppose \(y \not\in \mathbb{P}, y = p_1\times p_2\cdots p_n\), and \(x > p_i \cdots p_k\). Then \(\large\frac{x}{y} \normalsize =\) \(\large\frac{1}{y / p_i \cdots p_k} \normalsize +\) \(\large\frac{x - p_i \cdots p_k}{y}\).

  3. Express \(\large\frac{a}{b} \normalsize = \large\frac{1}{by} \normalsize +\) \(\large\frac{x}{y}\) according to step 1, or if step 2 applies, use

    \(\large\frac{a}{b} \normalsize = \large\frac{1}{by} \normalsize +\) \(\large\frac{1}{y / p_i \cdots p_k} \normalsize +\) \(\large\frac{x - p_i \cdots p_k}{y}\)

  4. For either \(x > 1\) or \(x - p_i \cdots p_k > 1\), set new values of either \(a = x\) or \(a = x - p_i \cdots p_k\) and \(b = y\); then iterate steps 1 through 3.

  5. Combine all unary fractions into a sum expressing the Egyptian fraction result.
